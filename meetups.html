Meetups

Meetups are an integral part of the Colombo Camps Community as sessions are frequently held on different aspects in Software Development. The Colombo Camps Community host meetups on areas such as Software Architecture, Security and Test Automation etc. where the interest groups get together to discuss one or few important topics.





Colombo Architecture Meetup

Colombo architecture meetup is a community event among professionals who involved in software design and development. This is an open event for software architects and technical leads in the industry. This event is basically to discuss about the latest trends and commonly faced challenges in software architecture. Highly experienced software professionals conduct sessions on topics selected by the community. This event also include some case studies and workshops.

Official Website : www.meetup.com/Colombo-Architecture-Meetup |


Colombo Selenium Meetup

The Colombo Selenium Meet-up is an open forum where Selenium users of all skill levels can come to communicate, learn, network,or just have a good time. We are meeting once a month and generally have speakers from major software and web companies in the area, which will be followed by Q&A session. Our focus is in Selenium and topics extend through related areas of interest, such as Agile testing, Test-Driven Development, Continuous Integration, Continuous Deployment, building maintainable automated test scripts and much more.

Official Websites : www.meetup.com/colombo-selenium | seshell.org



Colombo Security Meetup

Colombo Security Meetup is the srilankan chapter of OWASP. This is the first ever information security meetup in srilanka, where mainly focus on Application Security. Colombo Security Meetup is free and open to everyone, membership is not required to do so. Colombo Security Meetup consider as the place where meet Srilankan information security professionals and discuss application security trends. We have meetups in quarter base and generally have speakers from information security industry as well as academia. In meetups mainly we discuss about latest application security trends, testing environments, security tools, guidelines and checklists which help software companies to produce secure codes. Colombo Security Meetup is a fantastic place to learn about application security, to netrwork and even to build your reputation as an expert


Official Website : www.meetup.com/Colombo-Security-Meetup |


Colombo Performance Testing Meetup

The Colombo PerfTest Meetup will bring together the local performance test engineers, architects, designers, developers and anyone who is interested in software applications performance. Our mission is to facilitate professionals in designing and developing faster software applications. This gathering will allow you to present the experience in application performance engineering, learn latest trends from experts and network with your peers.


Official Website : www.meetup.com/ColomboPerfTestMeetup |
